# Documentation

This directory contains project documentation, design decisions, and sprint artifacts.

## Contents

Place the following here as the project progresses:

- **Architecture Decision Records (ADRs)** — Document key technical decisions and their rationale
- **API Documentation** — Endpoint specs, request/response formats
- **Sprint Artifacts** — Sprint goals, retrospective notes, velocity charts
- **Design Mockups** — UI/UX wireframes and design files
- **Meeting Notes** — Standup summaries, sprint review notes

## Naming Convention

Use descriptive filenames with dates where relevant:

```
docs/
├── adr/
│   └── 001-use-django-rest-framework.md
├── api/
│   └── endpoints.md
├── sprints/
│   └── sprint-1-goals.md
└── design/
    └── dashboard-wireframe.md
```
